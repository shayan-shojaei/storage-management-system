# Restaurant Storage Management System

## Description

A Nest.js API for a restaurant storage management system.

## Installation

```bash
$ yarn install
```

## Database Seeder

```bash
# clear database
$ yarn seeder:clear

# import data
$ yarn seeder:import

# clear and import
$ yarn seeder:reset
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
